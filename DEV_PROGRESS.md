## 一、项目概述

### 1.1 项目名称
百里教学管理系统

### 1.2 项目类型
小程序 + 网页管理后台

### 1.3 核心技术栈

**后端：**
- Express.js 4.x
- MongoDB + Mongoose
- JWT 认证
- Node.js 原生模块

**前端：**
- 原生 JavaScript (ES6+)
- HTML5 + CSS3
- 模块化架构
- 事件委托模式

### 1.4 项目结构

```
/workspace
├── backend/                    # 后端项目
│   ├── config/                 # 配置文件
```

## 五、技术亮点

### 5.1 数据访问层工厂模式
```javascript
// utils/db.js
const getDB = () => {
  if (process.env.USE_MEMORY_DB === 'true') {
    return require('./memoryDB');
  }
  return require('./models');
};
```

### 5.2 统一响应格式
```javascript
// middleware/response.js
res.success = (data, message = '操作成功') => { ... }
res.error = (message, code, details) => { ... }
```

### 5.3 积分系统
- 作业完成积分
- 打卡连续奖励
- 分享朋友圈额外积分
- 教师手动调整

### 5.4 安全防护
- JWT Token 认证
- 速率限制（认证接口 20次/15分钟，通用接口 100次/分钟）
- XSS 防护（escapeHtml/escapeAttr）
- 输入验证（express-validator）

## 六、已知限制

### 6.1 内存数据库模式
当前使用 `memoryDB.js` 模拟 MongoDB：
- 数据存储在内存中，服务重启后丢失
- 适用于开发/测试环境
- 生产环境需切换到真实 MongoDB

### 6.2 文件上传
- 前端支持文件上传数据结构
- 后端预留了 multer 中间件
- 实际文件存储需配置（本地/OSS）
